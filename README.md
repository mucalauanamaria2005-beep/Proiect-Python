# Proiect Python

recomandare filme

 Sistem de recomandare a filmelor



 Descriere

Acest proiect reprezintă un sistem de recomandare a filmelor dezvoltat în Python, folosind framework-ul Django. Aplicația oferă utilizatorilor sugestii de filme pe baza unui film preferat introdus și a unui criteriu de asemănare ales.







 Scopul proiectului

Scopul acestui proiect este de a demonstra utilizarea limbajului Python pentru dezvoltarea unei aplicații web, precum și implementarea unui algoritm simplu de recomandare bazat pe similaritatea dintre filme.







 Funcționalități

\- Introducerea unui titlu de film preferat

\- Alegerea unui criteriu de recomandare:

  - General

  - Gen

  - Temă

  - Actori

  - Regizor

\- Generarea unei liste de filme recomandate

\- Afișarea selecției utilizatorului înaintea recomandărilor







 Cum funcționează aplicația

Aplicația utilizează fișiere CSV care conțin informații despre filme (descriere, genuri, actori, regizor). Pe baza acestor date, se calculează similaritatea dintre filme folosind tehnici de procesare a textului și cosine similarity, iar utilizatorului îi sunt afișate cele mai relevante recomandări.
